# Return Home

A component for navigating back to the home page

##### Import

```js
import { ReturnHome } from '/components';
```

##### Usage

```jsx
<ReturnHome>
    <ChildComponent />
</ReturnHome>
```

##### Required props

| Name       | Type              | Description                           |
| ---------- | ----------------- | ------------------------------------- |
| `children` | `React.ReactNode` | Child component to be rendered inline |
