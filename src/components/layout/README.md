# Layout

A layout component for the pages of the app

##### Import

```js
import { Layout } from '/components';
```

##### Usage

```jsx
<Layout>
    <ChildComponent />
</Layout>
```

##### Required props

| Name       | Type              | Description                              |
| ---------- | ----------------- | ---------------------------------------- |
| `children` | `React.ReactNode` | Child component to be rendered in layout |
